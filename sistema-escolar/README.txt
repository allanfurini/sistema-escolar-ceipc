SISTEMA ESCOLAR CEIPC - VERSÃO 2

O que esta versão faz:
- cadastro de turmas
- cadastro de alunos com campos extras de responsáveis/documentos
- cadastro de disciplinas
- vinculação de disciplinas específicas para cada turma
- diário / boletim completo por aluno
- 4 bimestres na mesma tela
- até 4 notas por bimestre
- faltas por bimestre
- média automática por bimestre
- média anual automática
- cores: azul para média >= 7 e vermelho para média < 7
- boletim em página própria para impressão
- busca ampla por aluno, pai, mãe, código e telefone

Como usar:
1) cadastre as turmas
2) cadastre as disciplinas
3) vá em cada turma e marque as disciplinas daquela turma
4) cadastre os alunos
5) vá em Diário / Boletim
6) escolha turma e aluno
7) digite as notas e faltas
8) salve
9) abra o boletim para imprimir

Observação importante:
- login ainda não foi implementado nesta versão
- edição de cadastro ainda será melhorada depois
- esta versão prioriza o lançamento rápido das notas e a impressão do boletim
