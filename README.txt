SISTEMA ESCOLAR CEIPC - VERSÃO INICIAL

O que esta versão faz:
- cadastro de turmas
- cadastro de alunos
- vinculação simples do aluno à turma
- cadastro de disciplinas
- lançamento de notas por bimestre
- listagem de notas com filtro por turma

Como rodar localmente:
1) abra o terminal dentro da pasta sistema-escolar
2) execute:
   pip install -r requirements.txt
3) execute:
   python app.py
4) abra no navegador:
   http://127.0.0.1:5000

Observação:
- o banco escola.db será criado automaticamente na primeira execução
- a próxima etapa será boletim em layout parecido com o sistema antigo mostrado nos prints
